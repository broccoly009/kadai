# レポート課題３

表準画像「penguin.jpg」を使用する。


ORG=imread('penguin.jpg'); % 原画像の入力

ORG= rgb2gray(ORG); % カラー画像を白黒濃淡画像へ変換

imagesc(ORG); colormap(gray); colorbar; % 画像の表示
